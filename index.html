Sentinel \u2014 risk copilot for leveraged crypto portfolios. Built for OKX.AI Genesis Hackathon.

Features:

  • Real-time liquidation distance via live OKX pricing
  • Funding rate monitoring \u2014 automatic pull from OKX
  • Portfolio Health Score (0-100) \u2014 unified risk metric
  • Maintenance margin via OKX position-tiers API
  • Simulator \u2014 play with symbol, size, direction, leverage
  • Stress Twin \u2014 4 crisis scenarios, cross or isolated mode
  • Multi-position tracking with aggregates
  • P&L (unrealized) per position
  • Telegram alerts with snooze, recovery, custom intervals
  • OKX API import for real positions

Sentinel is a risk copilot for leveraged crypto portfolios, built for the OKX.AI Genesis Hackathon.

Features:

  • Real-time liquidation distance with live OKX pricing
  • Funding rate monitoring — automatic pull from OKX
  • Portfolio Health Score (0-100) — unified risk metric across all positions
  • Real maintenance margin via OKX position-tiers API
  • What-If slider — adjust leverage, see instant impact on liquidation
  • Stress Twin — simulate 4 crisis scenarios (including +10.10 crash)
  • Multi-position tracking with aggregate analytics
  • P&L tracking (unrealized)
  • Telegram alerts with configurable threshold
  • OKX API import for real positions

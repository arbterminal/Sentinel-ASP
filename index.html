Sentinel — risk copilot for leveraged crypto portfolios. Built for OKX.AI Genesis Hackathon.

Features:

  • Real-time liquidation distance via live OKX pricing
  • Funding rate monitoring — automatic pull from OKX
  • Portfolio Health Score (0-100) — unified risk metric
  • Maintenance margin via OKX position-tiers API
  • What-If Simulator — play with symbol, size, direction, leverage
  • Stress Twin — 4 crisis scenarios
  • Multi-position tracking with aggregates
  • P&L (unrealized) per position
  • Telegram alerts with snooze, recovery notification, custom intervals
  • OKX API import for real positions
